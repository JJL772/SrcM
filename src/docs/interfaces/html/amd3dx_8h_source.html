38  MACRO FORMAT
39  ============
40  This file contains inline assembly macros that
41  generate AMD-3D instructions in binary format.
42  Therefore, C or C++ programmer can use AMD-3D instructions
43  without any penalty in their C or C++ source code.
44 
45  The macro's name and format conventions are as follow:
46 
47 
48  1. First argument of macro is a destination and
49  second argument is a source operand.
50  ex) _asm PFCMPEQ (mm3, mm4)
51  | |
52  dst src
53 
54  2. The destination operand can be m0 to m7 only.
55  The source operand can be any one of the register
56  m0 to m7 or _eax, _ecx, _edx, _ebx, _esi, or _edi
57  that contains effective address.
58  ex) _asm PFRCP (MM7, MM6)
59  ex) _asm PFRCPIT2 (mm0, mm4)
60  ex) _asm PFMUL (mm3, _edi)
61 
62  3. The prefetch(w) takes one src operand _eax, ecx, _edx,
63  _ebx, _esi, or _edi that contains effective address.
64  ex) _asm PREFETCH (_edi)
65 
66  For WATCOM C/C++ users, when using #pragma aux instead if
67  _asm, all macro names should be prefixed by a p_ or P_.
68  Macros should not be enclosed in quotes.
69  ex) p_pfrcp (MM7,MM6)
70 
71  NOTE: Not all instruction macros, nor all possible
72  combinations of operands have been explicitely
73  tested. If any errors are found, please report
74  them.
75 
76  EXAMPLE
77  =======
78  Following program doesn't do anything but it shows you
79  how to use inline assembly AMD-3D instructions in C.
80  Note that this will only work in flat memory model which
81  segment registers cs, ds, ss and es point to the same
82  linear address space total less than 4GB.
83 
84  Used Microsoft VC++ 5.0
85 
86  #include <stdio.h>
87  #include "amd3d.h"
88 
89  void main ()
90  {
91  float x = (float)1.25;
92  float y = (float)1.25;
93  float z, zz;
94 
95  _asm {
96  movd mm1, x
97  movd mm2, y
98  pfmul (mm1, mm2)
99  movd z, mm1
100  femms
101  }
102 
103  printf ("value of z = %f\n", z);
104 
105  //
106  // Demonstration of using the memory instead of
107  // multimedia register
108  //
109  _asm {
110  movd mm3, x
111  lea esi, y // load effective address of y
112  pfmul (mm3, _esi)
113  movd zz, mm3
114  femms
115  }
116 
117  printf ("value of zz = %f\n", zz);
118  }
