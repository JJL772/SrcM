194 // Purpose: CLocalizedStringArg<> is a class that will take a variable of any
195 // arbitary type and convert it to a string of whatever character type
196 // we're using for localization (locchar_t).
197 //
198 // Independently it isn't very useful, though it can be used to sort-of-
199 // intelligently fill out the correct format string. It's designed to be
200 // used for the arguments of CConstructLocalizedString, which can be of
201 // arbitrary number and type.
202 //
203 // If you pass in a (non-specialized) pointer, the code will assume that
204 // you meant that pointer to be used as a localized string. This will
205 // still fail to compile if some non-string type is passed in, but will
206 // handle weird combinations of const/volatile/whatever automatically.
207 // --------------------------------------------------------------------------
208 
209 // The base implementation doesn't do anything except fail to compile if you
210 // use it. Getting an "incomplete type" error here means that you tried to construct
211 // a localized string with a type that doesn't have a specialization.
212 template < typename T >
213 class CLocalizedStringArg;
