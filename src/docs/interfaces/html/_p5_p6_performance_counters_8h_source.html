13 //---------------------------------------------------------------------------
14 // Format of the performance event IDs within this header file in case you
15 // wish to add any additional events that may not be present here.
16 //
17 // BITS 0-8 Unit Mask, Unsed on P5 processors
18 // BIT 9 Set if event can be set on counter 0
19 // BIT 10 Set if event can be set on counter 1
20 // BITS 11-15 Unused Set to zero
21 // BITS 16-23 Event Select ID, Only bits 16-21 used on P5 Family
22 // BITS 24-27 Unused set to zero
23 // BITS 28-32 Process family that the event belong to, as returned by
24 // the CPUID instruction.
