23 //
24 // These classes pass all filesystem interface calls through to another filesystem
25 // interface. They can be used anytime you want to override a couple things in
26 // a filesystem. VMPI uses this to override the base filesystem calls while
27 // allowing the rest of the filesystem functionality to work on the master.
28 //
