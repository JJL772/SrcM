14 // This class is used to virtualize a ConVar's value, so the client can restrict its
15 // value while connected to a server. When using this across modules, it's important
16 // to dynamic_cast it to a ConVar_ServerBounded or you won't get the restricted value.
17 //
18 // NOTE: FCVAR_USERINFO vars are not virtualized before they are sent to the server
19 // (we have no way to detect if the virtualized value would change), so
20 // if you want to use a bounded cvar's value on the server, you must rebound it
21 // the same way the client does.
