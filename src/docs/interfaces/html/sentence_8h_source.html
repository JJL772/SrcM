140 // A sentence can be closed captioned
141 // The default case is the entire sentence shown at start time
142 //
143 // "<persist:2.0><clr:255,0,0,0>The <I>default<I> case"
144 // "<sameline>is the <U>entire<U> sentence shown at <B>start time<B>"
145 
146 // Commands that aren't closed at end of phrase are automatically terminated
147 //
148 // Commands
149 // <linger:2.0> The line should persist for 2.0 seconds beyond m_flEndTime
150 // <sameline> Don't go to new line for next phrase on stack
151 // <clr:r,g,b,a> Push current color onto stack and start drawing with new
152 // color until we reach the next <clr> marker or a <clr> with no commands which
153 // means restore the previous color
154 // <U> Underline text (start/end)
155 // <I> Italics text (start/end)
156 // <B> Bold text (start/end)
157 // <position:where> Draw caption at special location ??? needed
158 // <cr> Go to new line
159
