AppendElementToMatrix - same as setting the element, except only works when all calls happen in top to bottom left to right order, end you have to call FinishedAppending when done. For normal matrix classes this is not different then SetElement, but for CSparseMatrix, it is an accelerated way to fill a matrix from scratch.
