8 //
9 // VMatrix always postmultiply vectors as in Ax = b.
10 // Given a set of basis vectors ((F)orward, (L)eft, (U)p), and a (T)ranslation,
11 // a matrix to transform a vector into that space looks like this:
12 // Fx Lx Ux Tx
13 // Fy Ly Uy Ty
14 // Fz Lz Uz Tz
15 // 0 0 0 1
16 
17 // Note that concatenating matrices needs to multiply them in reverse order.
18 // ie: if I want to apply matrix A, B, then C, the equation needs to look like this:
19 // C * B * A * v
20 // ie:
21 // v = A * v;
22 // v = B * v;
23 // v = C * v;
24 //=============================================================================
