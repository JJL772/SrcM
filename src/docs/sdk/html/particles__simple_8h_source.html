20 // CParticleEffect is the base class that you can derive from to make a particle effect.
21 // These can be used two ways:
22 //
23 // 1. Allocate a CParticleEffect-based object using the class's static Create() function. This gives
24 // you back a smart pointer that handles the reference counting for you.
25 //
26 // 2. Contain a CParticleEffect object in your class.
27 // ------------------------------------------------------------------------------------------------ //

161 //
