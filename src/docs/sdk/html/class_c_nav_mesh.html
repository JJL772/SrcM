void CNavMesh::PostProcessCliffAreas ( )

Temp way to mark cliff areas after generation without regen'ing. Any area that is adjacent to a cliff gets marked as a cliff. This will leave some big areas marked as cliff just because one edge is adjacent to a cliff so it's not great. The code that does this at generation time is better because it ensures that areas next to cliffs don't get merged with no-cliff areas.
