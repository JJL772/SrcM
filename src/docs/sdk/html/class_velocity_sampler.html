Detailed Description

Class to store a sampled history of velocity for an object – used for certain sound calculations.

Although this contains only one sample for now, it exists as an interface so as to make simpler the possibility of moving to a ring buffer implementation in the future.

The "sample rate" variable is not nominal: it should be used to specify the ClientThink() interval.

Be sure to use the beginSampling() function for the first sample, and addSample() thereafter: this will be relevant and necessary for a ring buffer implementation (which will have to perform certain initialization).
